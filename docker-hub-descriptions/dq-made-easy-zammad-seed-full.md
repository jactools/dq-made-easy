# dq-made-easy Zammad Seed

Repo-managed seed image that prepares Zammad support data and demo content for
the support profile.

## Features

- Support/demo data seed assets
- Repo-owned bootstrap flow for the Zammad stack
- Keeps support fixtures alongside the platform source tree

## Quick Start

```bash
docker pull jacbeekers/dq-zammad-seed:latest
```

## Tags

- `latest` - most recent build
- `0.11.5-<hash>` - content-addressed release tag used by the build scripts

## Usage

Use this image when bootstrapping the support deployment profile or when
refreshing Zammad demo fixtures.
