# Repository Copilot Instructions

This repository supplies a small hierarchy of Copilot instruction files. The canonical location for repository instructions is `.github/copilot/`.

Primary files (canonical):
- `.github/copilot/01-general.md` — high-level rules (file sizes, naming, layering, test-proof)
- `.github/copilot/02-fastapi-sqlalchemy.md` — FastAPI + SQLAlchemy conventions
- `.github/copilot/03-testing.md` — test tooling, coverage, arm64 guidance
- `.github/copilot/04-database.md` — migrations and DDL guidance (Alembic first)
- `.github/copilot/05-versioning.md` — version bump & docs rules
- `.github/copilot/06-internal-service-contracts.md` — env, URL, and trust-bundle contracts

Use these files as authoritative guidance for code edits, test runs, and CI-related changes. If a rule conflicts with an explicit developer or system instruction, raise the conflict to the user.
